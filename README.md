 Marion Tumnai Portfolio

A modern and responsive personal portfolio website built with React and Vite.  
This project showcases my skills, projects, and contact information as a frontend developer.


 Tech Stack

- React
- JavaScript (ES6+)
- HTML5
- CSS3
- Git & GitHub


 Features

- Responsive design (mobile, tablet, desktop)
- Clean and modern UI
- Custom favicon and personal branding
- Project showcase section
- Contact section
- Smooth navigation



Installation

1. Clone the repository
   git clone https://github.com/mariontamnai/new-portfolio.git

2. Navigate into the project folder
   cd marion-portfolio

3. Install dependencies
   npm install

4. Start the development server
   npm run dev



 Project Purpose

This portfolio was built to:
- Showcase my frontend development skills
- Present my projects in a professional way
- Practice modern React development
